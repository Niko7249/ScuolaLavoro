Le slide sono state create con l'utilizzo di diverse tecnologie open
source.

1) Emacs - Org-mode

Emacs è stato già da me utilizzato e possiamo considerarlo un text
editor estensibile e personalizzabile, esistono infatti un sacco di

2) Reveal.js - html presentazione framework

Reveal.js è un framework html open source e permette di creare queste
tipologie di presentazioni.
